Welcome to GRNBoost's documentation
===================================

GRNboost_ is an `Apache Spark`_ library for scalable inference of gene regulatory networks from next generation sequencing data.
It is written in the Scala_ programming language and freely `available on Github`_.

.. _GRNBoost: https://github.com/aertslab/GRNBoost
.. _Scala: https://www.scala-lang.org/
.. _Apache Spark: https://spark.apache.org/
.. _available on Github: https://github.com/aertslab/GRNBoost
